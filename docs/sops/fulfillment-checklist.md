# Fulfillment Checklist — Invictus AI
## Per-Deliverable Quality Standards

---

## 1. Website Deliverable

### Design & Layout
- [ ] Homepage with clear value proposition above the fold
- [ ] Services/offerings page with detailed descriptions
- [ ] About page with team/business story
- [ ] Contact page with form, map, phone, email, WhatsApp
- [ ] Testimonials/reviews section (minimum 3)
- [ ] Call-to-action on every page (Book Now / Call / WhatsApp)
- [ ] Professional color scheme matching brand
- [ ] Consistent typography and spacing
- [ ] High-quality images (no pixelated/stock watermarked)
- [ ] Footer with business info, social links, legal pages

### Content Quality
- [ ] No spelling or grammar errors
- [ ] No placeholder text (Lorem ipsum, "Your text here")
- [ ] Business name consistent everywhere
- [ ] Phone number correct and clickable (tel: link)
- [ ] Email correct and clickable (mailto: link)
- [ ] Address correct and linked to Google Maps
- [ ] WhatsApp link working with pre-filled message
- [ ] All service descriptions are accurate and approved by client
- [ ] Pricing (if shown) is current and approved

### SEO Fundamentals
- [ ] Unique title tag per page: "[Service] in [City] | [Business Name]"
- [ ] Meta description per page (150-160 chars, includes city + service)
- [ ] H1 tag on every page (only one per page)
- [ ] Image alt tags on all images (descriptive, include keywords)
- [ ] Schema markup: LocalBusiness with name, address, phone, hours
- [ ] XML sitemap generated and accessible
- [ ] robots.txt configured
- [ ] Google Search Console verified
- [ ] Google Analytics 4 installed and tracking
- [ ] Canonical URLs set

### Technical Performance
- [ ] Lighthouse Performance score > 80
- [ ] Lighthouse Accessibility score > 85
- [ ] Lighthouse SEO score > 90
- [ ] Page load time < 3 seconds on 4G
- [ ] Mobile responsive: tested on iPhone SE, iPhone 14, Samsung Galaxy
- [ ] Tablet responsive: tested on iPad
- [ ] All links working (no 404s) — run link checker
- [ ] All forms submit correctly + send notification
- [ ] SSL certificate active (HTTPS)
- [ ] No console errors in browser developer tools
- [ ] Images optimized (WebP where possible, max 200KB each)
- [ ] Lazy loading on below-fold images

### Legal & Compliance
- [ ] Privacy policy page (or link)
- [ ] Cookie consent banner (if applicable)
- [ ] Terms of service (if applicable)
- [ ] Professional disclaimers (medical/legal niches)

## 2. CRM Setup (Growth + Scale Tiers)

### Pipeline Configuration
- [ ] Lead pipeline created with stages matching business flow
- [ ] Stage automation: auto-move on form submission / appointment
- [ ] Lead source tracking configured
- [ ] Deal values set for revenue tracking

### Forms & Calendars
- [ ] Website inquiry form → CRM contact creation
- [ ] Appointment booking calendar configured
- [ ] Calendar synced with client's personal calendar (Google/Outlook)
- [ ] Booking confirmation email/SMS automated
- [ ] Business hours set correctly in calendar
- [ ] Buffer time between appointments configured

### Contact Management
- [ ] Contact fields: name, phone, email, service interest, source
- [ ] Tags system: niche, source, status, priority
- [ ] Duplicate detection enabled
- [ ] Client can view their own contacts/leads

### Notifications
- [ ] New lead notification → client's WhatsApp + email
- [ ] Appointment booked notification
- [ ] Form submission notification
- [ ] Missed call notification (if applicable)

## 3. Email & SMS Automations (Growth + Scale Tiers)

### Lead Nurture Sequence
- [ ] New lead → immediate auto-reply (< 60 seconds)
  - Email: "Thanks for reaching out! We'll contact you within [X] hours"
  - SMS/WhatsApp: "Hi [Name], thanks for contacting [Business]. We'll be in touch shortly!"
- [ ] Day 1: Follow-up if no response
- [ ] Day 3: Value-add email (tips, case study, testimonial)
- [ ] Day 7: Final follow-up with direct CTA

### Appointment Automations
- [ ] 24-hour reminder (SMS + email)
- [ ] 1-hour reminder (SMS only)
- [ ] Post-appointment follow-up (24h after)
- [ ] No-show follow-up (reschedule offer)

### Review Collection
- [ ] Post-service review request (24-48h after service)
- [ ] Google review link embedded in message
- [ ] Positive review → thank you response
- [ ] Negative review → alert to business owner for personal response
- [ ] Monthly review summary report

### Seasonal Campaigns (niche-specific)
**Dental:** Dental hygiene month, back-to-school checkups, holiday smile makeovers
**CA/Tax:** ITR deadlines, GST quarterly reminders, budget updates, tax-saving tips
**Education:** New batch announcements, exam prep, early bird enrollment
**Lawyers:** Legal awareness days, regulatory changes, free consultation offers

## 6. Ongoing Support Scope

### Starter Tier (No monthly retainer)
- 30-day bug fix support
- Email support only
- Response time: 48 hours
- No content updates included

### Growth Tier (₹8,000-12,000/mo)
- Priority support (response < 4 hours)
- Up to 4 content updates per month
- Monthly performance report
- CRM maintenance
- Automation monitoring
- Quarterly strategy call (30 min)

### Scale Tier (₹15,000-25,000/mo)
- Priority support (response < 2 hours)
- Unlimited content updates
- Weekly performance report
- Dedicated account manager
- CRM optimization + new automations
- Monthly strategy call (45 min)
- A/B testing on key pages
- Competitor monitoring
- Ad campaign management (if applicable)

## Quality Gate: Pre-Delivery Checklist

Before marking ANY client as "Ready for Review":
1. [ ] Run through entire website on mobile device (physical device, not browser emulator)
2. [ ] Submit every form and verify notifications arrive
3. [ ] Click every link on every page
4. [ ] Run Lighthouse audit (save report)
5. [ ] Check business name spelling one more time
6. [ ] Verify phone number by calling it
7. [ ] Send test appointment booking
8. [ ] Review all automated messages for accuracy
9. [ ] Get internal team sign-off
10. [ ] Only THEN share preview with client
